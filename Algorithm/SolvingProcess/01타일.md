### 문제
 **첫 번째 줄에 지원이가 만들 수 있는 길이가 N인 모든 2진 수열의 개수를 15746으로 나눈 나머지를 출력한다.**
 
#### 접근방법 1
> 쭉 적어보니 피보나치 수열이다.   
> 그리고 두 수를 더하고 나머지를 구하는것과 각각 나머지를 구한뒤 나머지 끼리 값이 같은 것을 이용하면 된다.

##### 코드
```cpp
#include <iostream>
using namespace std;

int memo[1000001];

int dp(int n) {
	if (n == 1) return 1;
	if (n == 2) return 2;
	if (memo[n] != 0) return memo[n];

	memo[n] = (dp(n - 1) + dp(n - 2)) % 15746;
	return memo[n];
}

int main() {
	memo[1] = 1;
	memo[2] = 2;

	int n;
	cin >> n;

	int ans = dp(n);
	cout << ans << endl;
}
```
